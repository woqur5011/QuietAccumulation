# 조용한 매집 대시보드

pykrx 기반 한국 주식 외인·기관 수급 데이터 자동 수집 + Streamlit 대시보드

---

## 파일 구조

```
QuietAccumulation/
├── collector.py       # pykrx 데이터 수집 (1회 실행 또는 scheduler에서 호출)
├── scheduler.py       # APScheduler — 1시간/6시간/1일 주기 자동 수집
├── dashboard.py       # Streamlit 대시보드
├── watchlist.json     # 관심 종목 목록 (없으면 코스피 상위 50개 자동)
├── requirements.txt
└── data/
    └── snapshot.csv   # 수집된 최신 스냅샷 (자동 생성)
```

---

## 설치

```bash
pip install -r requirements.txt
```

---

## 실행

### 1) 데이터 1회 수집
```bash
python collector.py
```

### 2) 스케줄러 실행 (백그라운드 상시 수집)
```bash
python scheduler.py
```

수집 주기:
| 구분 | 시간 |
|------|------|
| 장중 1시간 | 평일 09:00 ~ 15:00 (매 정시) |
| 장외 6시간 | 평일 16:00, 22:00 |
| 일일 스냅샷 | 평일 08:50 (장 시작 전) |

### 3) 대시보드 실행
```bash
# 터미널 2개를 열어 동시 실행 권장
# 터미널 1:
python scheduler.py

# 터미널 2:
streamlit run dashboard.py
```
브라우저에서 `http://localhost:8501` 접속

## 컬럼 설명

| 컬럼 | 설명 |
|------|------|
| 외인연속매수일 | 외국인 순매수가 연속으로 이어진 거래일 수 |
| 기관연속매수일 | 기관 순매수가 연속으로 이어진 거래일 수 |
| 외인매수(억) | 최근 거래일 외국인 순매수금액 (억 원) |
| 기관매수(억) | 최근 거래일 기관 순매수금액 (억 원) |
| 중간매도기간(외/기) | 최근 연속 순매도 기간 (매집 중 눌림 판단) |
| 종가 | 최근 거래일 종가 |
| TV5/20 | 최근 5일 평균거래량 / 최근 20일 평균거래량 (거래량 폭발 지표) |
| 52주위치(%) | (현재가 - 52주 저가) / (52주 고가 - 52주 저가) × 100 |
| 재무등급 | ROE·PBR·PER 기반 간단 등급 (S/A/B/C/D) |
